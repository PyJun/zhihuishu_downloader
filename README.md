# 知到智慧树视频课程下载器（Win电脑版）
知到智慧树视频课程下载工具，一键下载知到智慧树上面的视频课件资料

工具获取地址：https://lanzouw.com/s/course-downloader

## 安装教程
从蓝奏云网盘下载最新版学无止下载器后，点击确认一路安装即可

【如果电脑安装杀毒软件，建议关闭杀毒软件后再下载安装，以免误报拦截】
